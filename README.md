## 📚 학습 목적

React의 전체적인 구조와 흐름을 다시 정리하고,  
기초 개념을 단단하게 다지기 위해 학습했다.

이미 알고 있는 내용도 많았지만,  
개념을 체계적으로 정리하고 복습하는 데 의미를 두었다.

## 🔎 책의 구성 및 흐름

### 1️⃣ 도입부 – JavaScript 기초

초반부에서는 React를 이해하기 위해 필요한  
기본적인 JavaScript 문법과 개념들을 다룬다.
이미 익숙한 내용이었지만,  
React 학습의 기반이 되는 부분을 다시 점검할 수 있었다.

---

### 2️⃣ 중반부 – React 기본 개념

본격적으로 React의 핵심 개념들을 다룬다.

- 컴포넌트 구조
- Props
- State
- 이벤트 처리
- 조건부 렌더링
- 리스트 렌더링
- Hooks (useState, useEffect 등)

내용이 깊게 들어가지는 않지만,  
React의 전체적인 구조를 이해하기에는 충분한 구성이다.

---

## 💭 느낀 점

내 기준에서는 비교적 쉽게 느껴졌고,  
새로운 내용을 배우기보다는 **이미 알고 있던 개념을 정리하고 상기하는 과정**에 가까웠다.

하지만 다음과 같은 점에서 의미가 있었다:

- React의 기본 흐름을 다시 구조적으로 정리할 수 있었다.
- 개념을 “알고 있는 상태”에서 “설명할 수 있는 상태”로 정리하는 데 도움이 되었다.
- 이후 심화 학습으로 넘어가기 위한 기반을 다지는 시간이었다.
